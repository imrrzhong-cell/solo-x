**Markdown 渲染规范：**
- h2: Noto Serif SC, 1.5rem, 带左侧 sage 竖线装饰
- h3: Noto Serif SC, 1.2rem
- p: 正文 0.92rem, 行高 1.8, 字重 300
- code: 内联代码使用 warm 背景
- blockquote: 左侧 sage 竖线 + 斜体
- img: 圆角 4px, 最大宽度 100%
- 付费墙：部分内容模糊遮罩 + CTA

## 4. 交互说明

### 4.1 全局交互

| 元素 | 触发 | 效果 | 持续时间 |
|------|------|------|----------|
| Nav 背景 | 页面滚动 | 始终固定顶部，半透明白底 + blur(10px) | 常驻 |
| Nav 链接 | hover | 文字颜色过渡 char3 → sage2 | 0.2s |
| Nav CTA 按钮 | hover | 背景色过渡 sage2 → sage3 | 0.2s |
| 页面滚动 | scroll | smooth scrolling（html 全局）| 常驻 |

### 4.2 首页交互

| 元素 | 触发 | 效果 | 持续时间 |
|------|------|------|----------|
| Hero 元素 | 页面加载 | fadeIn 上移渐显，分4级延迟 | 1s + delay |
| Hero CTA (btn-sage) | hover | 背景色 sage2 → sage3 + 上移1px | 0.3s |
| Hero 文字链接 (btn-plain) | hover | 文字色 → sage2 + 下边框变色 | 0.2s |
| zen-card | hover | 背景色 off → warm + 边框色加深 | 0.3s |
| ct-item (内容卡片) | hover | 背景色 white → off | 0.3s |
| tl-item (工具卡片) | hover | 背景色 white → off + 边框色加深 | 0.3s |
| pc-btn (定价按钮) | hover | 背景色变为 sage2 + 文字变白 | 0.3s |
| pick 内 pc-btn | 默认 | 背景已是 sage2 + 白色文字 | 常驻 |
| pick 内 pc-btn | hover | 背景色 sage2 → sage3 | 0.3s |

### 4.3 文章列表交互

| 元素 | 触发 | 效果 | 持续时间 |
|------|------|------|----------|
| 分类标签 | click | 切换选中状态，过滤文章列表 | 即时 |
| 分类标签 | click | 下划线动画指示当前分类 | 0.2s |
| 文章卡片 | hover | 背景色变化 + 轻微阴影 | 0.3s |
| 搜索框 | focus | 边框色变为 sage + 扩展宽度 | 0.3s |
| 加载更多 | click | 新文章 fadeIn 追加到列表底部 | 0.5s |

### 4.4 文章详情交互

| 元素 | 触发 | 效果 | 持续时间 |
|------|------|------|----------|
| 目录导航 | click | 平滑滚动到对应锚点 | smooth |
| 目录高亮 | scroll | 当前阅读位置对应的目录项高亮 | 即时 |
| 代码块 | hover | 右上角显示「复制」按钮 | 即时 |
| 付费墙 | scroll | 滚动到付费内容时显示模糊遮罩 + CTA | 即时 |

| 返回按钮 | click | 返回文章列表，保持滚动位置 | 即时 |

### 4.5 通用动画规范

| 动画名 | 属性 | 用途 |
|--------|------|------|
| fadeIn | opacity: 0→1, translateY: 20px→0 | 页面元素入场，1s ease |
| delay-1 | 延迟 0.2s | 二级元素入场 |
| delay-2 | 延迟 0.4s | 三级元素入场 |
| delay-3 | 延迟 0.6s | 四级元素入场 |
| hover 过渡 | background-color / color / border-color / transform | 所有交互反馈，0.2s-0.3s |

## 5. 响应式断点行为表

### 5.1 断点定义

| 断点名 | 宽度范围 | 设备 |
|--------|----------|------|
| Mobile S | < 375px | iPhone SE |
| Mobile | 375px - 767px | 手机 |
| Tablet | 768px - 1023px | 平板 |
| Desktop | 1024px - 1439px | 笔记本/小屏桌面 |
| Desktop L | >= 1440px | 大屏桌面 |

### 5.2 各区域响应式行为

#### 导航栏 (nav)

| 断点 | 行为 |
|------|------|
| Desktop | 水平导航，全部菜单项可见，CTA 右侧 |
| Tablet | 水平导航，菜单项间距缩小 |
| Mobile | Logo + 汉堡菜单图标，点击展开全屏导航抽屉 |

#### Hero 区域

| 断点 | 行为 |
|------|------|
| Desktop | 双栏 grid（1fr 1fr），gap 5rem |
| Tablet | 双栏 grid，gap 3rem |
| Mobile | 单栏堆叠，左栏在上、右栏（卡片）在下 |

#### 内容网格 (ct-grid)

| 断点 | 行为 |
|------|------|
| Desktop | 3列 × 2行 |
| Tablet | 2列 × 3行 |
| Mobile | 1列 × 6行 |

#### 工具网格 (tools-scroll)

| 断点 | 行为 |
|------|------|
| Desktop | 4列 × 2行 |
| Tablet | 2列 × 4行 |
| Mobile | 2列 × 4行 或 1列 × 8行 |

#### 定价网格 (price-grid)

| 断点 | 行为 |
|------|------|
| Desktop | 3列 |
| Tablet | 3列（缩小间距）或 1列 × 3行 |
| Mobile | 1列 × 3行 |

#### Section 间距

| 断点 | padding |
|------|---------|
| Desktop | 4rem (top/bottom) × 6% (left/right) |
| Tablet | 3rem × 4% |
| Mobile | 2.5rem × 5% |

#### 最大内容宽度

| 元素 | 最大宽度 |
|------|----------|
| 全局内容 | max-width: 1100px |
| 文章正文 | max-width: 680px |
| 法律页面 | max-width: 720px |

### 5.3 字体缩放

| 元素 | Desktop | Mobile |
|------|---------|--------|
| hero-title | clamp(2.2rem, 5vw, 3.8rem) | clamp 自动缩放 |
| sec-title | 1.8rem | 1.4rem |
| 正文 | 0.78-0.92rem | 不变 |
| 导航链接 | 0.8rem | 0.9rem（移动端展开菜单）|

## 附录：技术实现备注

1. **框架建议**：静态站点生成器（Next.js / Astro / Hugo），Markdown 文章渲染
2. **CSS 方案**：延续 demo-6-zen.html 的 CSS Variables 体系，全局共享 `:root` 变量
3. **动画实现**：CSS animation + IntersectionObserver（滚动触发 fadeIn）
4. **文章渲染**：Markdown → HTML，需支持代码高亮、图片优化、付费墙
5. **图片策略**：WebP 格式 + 懒加载 + 响应式 srcset
6. **SEO**：每个页面独立 meta(title, description, og:image)，文章页需结构化数据
7. **性能目标**：Lighthouse Performance >= 90，FCP < 1.5s
